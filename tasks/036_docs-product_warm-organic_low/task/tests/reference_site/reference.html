Color Tokens

All color values are defined as CSS custom properties on :root. Reference them using the var(--token-name) syntax.

Terracotta

--color-terracotta

#C4654A

Sage

--color-sage

#7A9B76

Brown

--color-brown

#5C4033

Cream

--color-cream

#FDF6EC

CSS 
/* Primary palette tokens */
--color-terracotta: #C4654A;
--color-terracotta-dark: #A8503A;
--color-terracotta-light: #D4836A;

/* Sage family */
--color-sage: #7A9B76;
--color-sage-dark: #5F7E5B;
--color-sage-light: #A3BFA0;
--color-sage-pale: #D4E5D2;

/* Neutral backgrounds */
--color-cream: #FDF6EC;
--color-warm-white: #FFFCF7;

/* Text colors */
--color-text-primary: #3B2F2F;
--color-text-secondary: #6B5B4F;
--color-text-muted: #9A8A7C;

Typography

Four font families serve distinct roles. The type scale follows a 1.25 ratio from a 1rem base.

Display

Playfair Display

Used for headings (h1–h6), hero titles, and testimonial quotes. Provides editorial warmth.

--font-display

Body Text

Work Sans

Primary body font for paragraphs, descriptions, and general reading content.

--font-text

UI Elements

Space Grotesk

Used for buttons, navigation links, labels, tags, and interactive UI components.

--font-ui

Monospace

JetBrains Mono

Code snippets, inline code references, and technical identifiers.

--font-mono

CSS 
/* Type scale (1.25 ratio) */
--text-xs:   0.75rem;   /* 12px */
--text-sm:   0.875rem;  /* 14px */
--text-base: 1rem;      /* 16px */
--text-md:   1.125rem;  /* 18px */
--text-lg:   1.25rem;   /* 20px */
--text-xl:   1.5rem;    /* 24px */
--text-2xl:  1.875rem;  /* 30px */
--text-3xl:  2.25rem;   /* 36px */
--text-4xl:  3rem;      /* 48px */
--text-5xl:  3.75rem;   /* 60px */

/* Font weights */
--weight-normal:   400;
--weight-medium:   500;
--weight-semibold: 600;
--weight-bold:     700;

Buttons

Three button variants with two size modifiers. Combine the base .btn class with a variant and optional size class.

Button Variants

Primary Secondary Sage
Large Primary Small Secondary
HTML 
<!-- Primary button -->
<a class="btn btn--primary" href="#">Get Started</a>

<!-- Secondary outline button -->
<a class="btn btn--secondary" href="#">Learn More</a>

<!-- Large sage button -->
<a class="btn btn--sage btn--large" href="#">Explore</a>

<!-- Small button -->
<button class="btn btn--primary btn--small">Copy</button>

Cards

Flexible card component with optional icon, tag, title, description, meta, and link slots.

HTML 
<div class="card">
  <div class="card__icon card__icon--sage">
    <!-- SVG icon here -->
  </div>
  <span class="card__tag">Category</span>
  <h3 class="card__title">Card Title</h3>
  <p class="card__description">Description text.</p>
  <a class="card__link" href="#">
    Read more <span class="card__link-arrow">&rarr;</span>
  </a>
</div>

Code Snippets

Dark-themed code blocks with a macOS-style header, language label, and syntax highlighting via span classes.

HTML 
<!-- Code snippet structure -->
<div class="code-snippet">
  <div class="code-snippet__header">
    <div class="code-snippet__dots">
      <span class="code-snippet__dot code-snippet__dot--red"></span>
      <span class="code-snippet__dot code-snippet__dot--yellow"></span>
      <span class="code-snippet__dot code-snippet__dot--green"></span>
    </div>
    <span class="code-snippet__language">CSS</span>
  </div>
  <div class="code-snippet__body">
    <pre class="code-snippet__pre">...</pre>
  </div>
</div>

Syntax Highlight Classes

.code-snippet__keyword Properties, tags
.code-snippet__string String values, attributes
.code-snippet__number Numeric values
.code-snippet__comment Comments
.code-snippet__function Function names

Utility Classes

Layout helpers, grid utilities, and spacing shortcuts for rapid composition.

Grid Utilities

Pre-built grid layouts with 2, 3, or 4 columns. Responsive breakpoints collapse to single column on mobile.

.grid-2 · .grid-3 · .grid-4

Container

Centered max-width wrapper with horizontal padding. Narrow variant available for content-focused layouts.

.container · .container--narrow

Spacing Helpers

Margin-top and margin-bottom utility classes at common spacing intervals for quick vertical rhythm adjustments.

.mt-4 · .mt-8 · .mb-4 · .mb-12 …

HTML 
<!-- Centered container -->
<div class="container">...</div>

<!-- Narrow container -->
<div class="container container--narrow">...</div>

<!-- 3-column grid with top margin -->
<div class="grid-3 mt-8">
  <div>Column 1</div>
  <div>Column 2</div>
  <div>Column 3</div>
</div>

<!-- Text alignment & accessibility -->
<p class="text-center">Centered text</p>
<span class="sr-only">Screen reader only</span>
