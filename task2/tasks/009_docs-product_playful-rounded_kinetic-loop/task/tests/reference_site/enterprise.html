Frequently asked questions

Common questions from enterprise evaluation teams.

Where is my data stored?

All CloudNest Enterprise data is stored in EU data centres (Frankfurt, Amsterdam, Dublin). We never transfer data outside the EU unless you explicitly configure a non-EU region.

Can I use CloudNest on-premise?

Yes. Our Enterprise tier offers a fully self-hosted deployment option via Docker and Kubernetes, with the same feature set as our cloud offering.

What happens to my data if I cancel?

You can export all content as Markdown, HTML, or PDF at any time. After cancellation, data is retained for 30 days before permanent deletion, per GDPR requirements.

Do you sign custom DPAs and BAAs?

Absolutely. We provide a standard DPA with all enterprise contracts and can negotiate custom Data Processing Agreements and Business Associate Agreements for healthcare customers.

How does migration from Confluence work?

Our migration assistant connects directly to your Confluence instance via API. It preserves page hierarchy, attachments, and inline comments. Most teams complete migration in 1–3 days.
