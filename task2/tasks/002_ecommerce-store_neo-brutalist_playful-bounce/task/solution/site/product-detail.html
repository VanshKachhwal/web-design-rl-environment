What's Included

Everything you need to keep your Git repos healthy and tidy.

⚡

Branch Health Monitor

Instantly see which branches are stale, ahead, behind, or conflicting. Color-coded output makes triage a breeze.

📊

Commit Velocity Graph

ASCII-rendered sparklines show your commit frequency over the last 30 days. Spot slowdowns before they become blockers.

🧹

Auto-Clean Engine

Safely remove merged and stale branches with a single command. Dry-run mode previews changes before any deletion.

📬

Daily Health Reports

Schedule automated repo health summaries that pipe to stdout, a log file, or your team's Slack webhook.

🔌

CI/CD Integration

Drop GitPulse into GitHub Actions, GitLab CI, or any pipeline. JSON output mode makes parsing trivial.

⚙️

Zero-Config Setup

Install via npm, brew, or cargo. Works out of the box with sensible defaults. Optional .gitpulse.yml for power users.

Quick Start

Get GitPulse running in under 60 seconds. Three install methods, zero dependencies, instant results.

# via npm
npm install -g gitpulse-cli
# via homebrew
brew install terracli/tap/gitpulse
# via cargo
cargo install gitpulse
~/my-project $ gitpulse status
